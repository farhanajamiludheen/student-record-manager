# Student Record Manager

A student record management system built with HTML and CSS, with database design concepts based on SQL.

## About
This project was built to practice frontend development and database design thinking. It simulates how a real student management system works, with features inspired by SQL database operations like insert, update, delete and search.

## Features
- Add, edit and delete student records
- Search and filter students by name, ID or course
- Sort records by any column
- Grade status tracking (Pass / Average / Fail)
- Live statistics dashboard
- Export records to CSV

## Tech Stack
- HTML
- CSS
- SQL concepts (filtering, sorting, CRUD operations)
